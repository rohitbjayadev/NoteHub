# NoteHub

**NoteHub** is a minimal and efficient note-taking app built with Flutter. It allows users to create, edit, reorder, and manage notes with a simple and responsive interface. The app uses **Hive** for local storage, ensuring that all notes are saved securely and persist between sessions.

## Features
- Create and save notes locally  
- Drag-and-drop note reordering  
- Persistent storage using Hive  
- Responsive design with a clean dark UI  
- Auto-generated timestamps for each note  
- View full content of longer notes  

## Tech Stack
- **Framework:** Flutter  
- **Language:** Dart  
- **Local Storage:** Hive & Hive Flutter  
- **UI:** Material Design
